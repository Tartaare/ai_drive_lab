# Night Train

Un projet de visualisation 3D interactive avec React Three Fiber et Drei.

## Fonctionnalités

- Rendu 3D avec Three.js et React Three Fiber
- Helpers Drei pour des composants 3D avancés
- MeshReflectorMaterial pour les surfaces réfléchissantes
- Instances pour optimiser les performances
- Contrôles de scroll
- Menu F3 pour ajuster la hauteur, la réflexion, le flou et la rugosité du sol

## Installation

```bash
npm install
```

## Développement

```bash
npm start
```

L'application s'ouvrira sur [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Technologies

- React 18.0.0
- Three.js 0.139.2
- @react-three/fiber 8.0.12
- @react-three/drei 9.77.3
